# CS-370-8-2-Portfolio
What work did I do on this project? What code was given and what did I create?
For this project, I was given starter code that already set up the pirate game environment. This included how the agent moves, how rewards are given, and the basic structure for training the agent.
I wrote the main learning parts myself. I built the neural network, added the Q-learning logic, set up experience replay, and finished the training loop so the pirate agent could learn from its actions. I also adjusted settings like exploration to help the agent improve over time.

What do computer scientists do and why does it matter?
Computer scientists create programs and systems that solve problems. They build software, apps, and intelligent systems that people use every day. This matters because technology is used in almost every field, like healthcare, business, security, and communication. Good computer science helps make systems faster, smarter, and more reliable.

How do I approach a problem as a computer scientist?
I start by breaking the problem into smaller pieces. I figure out what the inputs and outputs should be and what the goal is. Then I build and test the solution step by step. If something does not work, I adjust it and test again. In this project, I tested different settings and improved the agent based on how it performed.

What are my ethical responsibilities to the end user and the organization?
I am responsible for making sure the systems I build are safe, reliable, and fair. I need to protect user data and make sure the system does not behave in harmful or biased ways. I also have a responsibility to the organization to create accurate, secure, and well-tested solutions. Ethical behavior helps build trust and ensures technology is used responsibly.
